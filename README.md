# FUTURE_ML_01 - Sales Forecasting Project

## 📊 Project Overview
This project focuses on predicting future sales using historical business data.  
It helps businesses understand trends, forecast demand, and make better decisions.

---

## 🎯 Objective
The goal is to build a machine learning model that can:
- Predict future sales
- Analyze trends and patterns
- Provide business-friendly insights

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🧹 Data Preparation
- Removed missing values
- Converted date column into proper format
- Sorted data based on time

---

## 📈 Data Visualization
- Sales trend over time
- Monthly sales analysis (seasonality)

---

## 🤖 Model Building
- Used Linear Regression
- Created time-based feature (days)
- Trained model on historical sales data

---

## 🔮 Forecasting
- Predicted sales for the next 30 days
- Visualized actual vs predicted sales

---

## 📊 Output
- Sales trend graph  
- Forecast graph  
- Future sales prediction table  

---

## 💼 Business Insights
- Sales show a steady increasing trend over time  
- Monthly patterns indicate seasonal demand  
- Forecast predicts continued growth  

### 📌 Recommendations:
- Increase inventory to avoid stock shortages  
- Prepare staffing for peak demand  
- Improve supply chain planning  

---

## 🚀 Conclusion
This project demonstrates how machine learning can be used to support real business decisions through data-driven forecasting.
